Placeholder Images Directory
=========================

This directory contains placeholder images for the Ahmad Web Solutions website.

Required Images:
- project1.jpg through project9.jpg (Gallery page projects)
- demo1.jpg through demo6.jpg (Demos page previews)

Recommended Image Sizes:
- Project images: 600x400px
- Demo previews: 800x500px

Image Guidelines:
- Use high-quality images with professional appearance
- Optimize file sizes for web performance
- Use descriptive alt text in HTML
- Consider using placeholder services like:
  * https://placeholder.com/
  * https://picsum.photos/
  * https://unsplash.com/

To replace placeholders:
1. Add your images to this directory
2. Update the src attributes in the HTML files
3. Ensure proper alt text for accessibility

Note: Current HTML files reference placeholder image paths.
Update these paths when adding actual images.
